# LandlyAI - AI-Powered Interview Preparation Platform

LandlyAI is a modern, full-stack web application that helps developers prepare for technical interviews using AI-generated questions and explanations. Create personalized interview prep sessions tailored to your role, experience level, and topics you want to focus on.
